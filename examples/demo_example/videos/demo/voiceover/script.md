# Videowright — Demo

## intro

Videowright. Build videos in the same HTML, CSS, and JavaScript you build apps with.

## feature-svg

Animated SVG, drawn with web standards. Lines stroke in, nodes appear — zero dependencies.

## feature-three

Three.js for 3D. A full WebGL scene, right inside a segment.

## feature-lottie

Lottie for designer-made motion. Export from After Effects, play inside a segment.

## feature-echarts

Real product data. ECharts, your charting library, your component library, your UI.

## feature-cards

Videos compound. Segments, components, transitions, styles — all top-level, all reusable.

## outro

Get started. Install Videowright, and let the agent build your first video.
